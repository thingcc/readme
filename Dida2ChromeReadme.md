【标题】 滴答清单 Web 收集助手 (TickTick / Dida365 Web Clipper)

【简介】
一键将网页化为待办！为滴答清单量身定制的浏览器 GTD 收集神器。

在日常浏览网页、查阅资料或阅读长文时，总会遇到需要稍后处理的网页或重要片段。通过本扩展，你可以彻底告别繁琐的“复制-切换标签-粘贴”流程，实现沉浸式的工作流。

✨ 核心功能

🖱️ 一键快速收集：点击浏览器工具栏图标，即可将当前网页的标题和链接直接发送至你的滴答清单收集箱。

✂️ 右键精准划线：选中网页中的任意文字，右键点击即可快速将其转化为待办任务的标题或备注。

⚙️ 丰富的高级配置：支持自定义默认保存的清单文件夹、标签（Tags）、优先级以及任务日期，满足各种定制化的高效管理需求。

⚡️ 极简且极速：纯原生 API 对接，无需后台常驻运行，丝般顺滑，不拖慢浏览器性能。

🔒 隐私与安全承诺
我们坚守“极客与干净”的代码哲学：本插件所有核心逻辑均在您的本地浏览器内运行。绝不收集、不追踪您的任何浏览记录或隐私数据。与滴答清单的数据交互采用官方安全的 OAuth 2.0 加密协议，让您的数字资产绝对安全。

📄 隐私权政策 (Privacy Policy)
最后更新日期：2026年5月

1. 简介
“滴答清单 Web 收集助手”（以下简称“本扩展”）深知隐私对您的重要性，并致力于保护您的个人数据安全。本隐私政策旨在透明、清晰地说明我们在您使用本扩展时，如何处理必要的信息。

2. 信息的处理与范围
本扩展坚持“最小权限原则”，仅为实现核心收集功能而处理以下必要数据：

网页上下文：当您主动触发收集功能时，本扩展会临时获取当前标签页的标题与 URL，或您手动高亮选中的文本，用于构建待办任务。

账户授权：本扩展使用安全的 OAuth 2.0 协议获取滴答清单的访问令牌（Token），以实现任务写入。本扩展绝不触碰、不保存您的账号密码。

3. 数据的存储与安全
绝对本地化：本扩展的运行代码及您的个性化配置项（如默认清单、偏好设置）均仅存储在您的本地浏览器中（通过 chrome.storage API）。

无第三方服务器：我们没有设立任何外部服务器来中转、接收或分析您的数据。您的任务数据直接经由 HTTPS 加密通道传输至滴答清单（TickTick）官方服务器。

零数据追踪：本扩展不包含任何数据埋点、广告追踪或第三方行为分析脚本（如 Google Analytics 等）。

4. 第三方服务声明
本扩展依赖滴答清单（TickTick / Dida365）的官方 API 提供服务。当任务成功发送后，您的数据存储与管理将受滴答清单官方隐私政策的约束。

5. 信息共享限制
我们承诺，在任何情况下均不会收集您的浏览历史，更不会将您的任何信息出售、交易或转让给任何第三方机构。

6. 您的权利与撤销
您可以随时在浏览器的扩展管理中禁用或卸载本扩展。您也可以随时在您的滴答清单账户的安全设置页，撤销对本扩展的 OAuth 访问授权，撤销后本扩展将立即失去写入数据的权限。
