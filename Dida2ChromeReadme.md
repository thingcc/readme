# 🚀 滴答清单 Web 收集助手 (TickTick Web Clipper)

**一款为滴答清单量身定制的 Chrome 浏览器 GTD 收集神器，一键将网页化为待办！**

---

## ✨ 核心功能 (Features)

- **🖱️ 一键快速收集**：点击浏览器工具栏图标，即可将当前网页的标题和链接直接发送至你的滴答清单收集箱。
- **✂️ 右键精准划线**：选中网页中的任意文字，右键点击即可快速将其转化为待办任务的标题或备注。
- **⚙️ 丰富的高级配置**：支持自定义默认保存的清单文件夹、标签（Tags）、优先级以及任务日期，满足各种定制化的高效管理需求。
- **⚡️ 极简极速体验**：纯原生 API 对接，无需后台常驻运行，丝般顺滑，不拖慢浏览器性能。

## 🔒 隐私政策 (Privacy Policy)

**生效日期：2026年5月**

我们深知隐私对您的重要性。本扩展程序（滴答清单 Web 收集助手）的设计初衷是纯粹的效率提升，严格遵循“最小权限”和“数据本地化”原则。我们在此郑重承诺：

### 1. 数据收集与使用
- **绝不追踪**：本扩展完全在您的本地浏览器内独立运行，**不收集、不记录、不共享**您的任何浏览历史或隐私数据。
- **按需处理**：仅在您主动触发收集功能时，临时获取当前标签页的标题与 URL，或您手动高亮的文本，用于构建待办任务。
- **安全授权**：使用官方安全的 OAuth 2.0 协议获取滴答清单的访问令牌（Token），绝不触碰或保存您的账号密码。

### 2. 权限声明
本插件仅申请了维持核心功能运转的最低必要权限：
- `storage` 权限：**仅用于在您的本地设备上保存高级配置项**（例如：默认清单、标签偏好）。这些配置数据仅保存在您本机的 Chrome 浏览器中。
- `activeTab` / `contextMenus` 权限：仅用于在您主动点击图标或右键菜单时，获取当前网页的上下文信息并执行收集动作。

### 3. 数据传输与第三方服务
- **无中间服务器**：我们没有设立任何用于接收或中转用户数据的外部服务器。您的任务数据直接经由 HTTPS 加密通道传输至滴答清单（TickTick / Dida365）官方服务器，受其官方隐私政策约束。
- **无第三方追踪**：本扩展不包含任何数据埋点、广告分发模块或第三方行为分析脚本（如 Google Analytics 等）。

### 4. 您的权利与撤销
您可以随时在浏览器的扩展管理中禁用或卸载本扩展。您也可以随时在您的滴答清单账户的安全设置页中，撤销对本扩展的 OAuth 访问授权，撤销后本扩展将立即失去写入数据的权限。
