# todo
一键将当前网页或者网页中选中内容，发送到滴答清单的插件，支持丰富的配置功能，欢迎使用。
# 隐私权政策

最后更新日期：2025年9月27日

1. 简介
一键滴答清单浏览器扩展（以下简称"本扩展"）尊重并保护所有使用服务用户的个人隐私权。本隐私权政策旨在说明我们如何收集、使用、存储和保护您的信息。

2. 信息收集
本扩展收集的信息仅限于必要的最小范围，以实现其核心功能：

当前网页信息：当您使用本扩展创建任务时，我们会收集当前标签页的标题和URL，用于创建滴答清单任务。

选中的文本：如果您通过右键菜单创建任务，我们会收集您选中的文本内容。

滴答清单账户信息：为了创建任务，我们需要访问您的滴答清单账户，包括必要的OAuth令牌。

3. 信息使用
收集的信息仅用于以下目的：

创建和管理滴答清单任务

提供扩展的基本功能

改善用户体验

4. 信息存储
所有数据都存储在本地或浏览器的同步存储中：

绝对承诺：所有代码都在浏览器中运行，不会发送到服务器，已经过Google官方审核。

本地存储：扩展设置和偏好存储在浏览器的本地存储中。

同步存储：如果您启用了浏览器同步功能，部分数据可能会存储在云端。

滴答清单服务：创建的任务数据存储在滴答清单的服务器上。

5. 数据安全
我们采取合理的安全措施来保护您的信息：

使用安全的OAuth 2.0协议进行身份验证。

所有与滴答清单API的通信都通过HTTPS加密。

不收集不必要的个人身份信息。

6. 第三方服务
本扩展使用滴答清单的API服务。您的数据将受到滴答清单隐私政策的约束。建议您阅读滴答清单的隐私政策以了解他们如何处理您的数据。

7. 信息共享
我们不会将您的信息出售、交易或以其他方式转让给第三方，除非：

为了提供您要求的服务而必要的共享

遵守法律法规要求

保护我们的权利、财产或安全

8. 您的权利
您有权：

访问我们持有的关于您的信息

要求更正不准确的信息

要求删除您的数据

撤回对数据处理的同意

9. 政策变更
我们可能会不时更新本隐私政策。任何变更都会在本页面发布，并更新"最后更新日期"。

10. 联系我们
如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

GitHub Issues: https://github.com/thingcc/todo/issues

项目主页: https://todo.thing.cc
