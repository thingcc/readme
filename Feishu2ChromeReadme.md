# 🚀 飞书 OKR 看板增强 (Feishu OKR Mate)

**一款专为飞书 OKR 打造的 Chrome 浏览器增强插件，旨在优化界面排版，提供更加沉浸和高效的目标管理体验。**

---

## ✨ 核心功能 (Features)

- **🖥️ 紧凑排版模式**：去除原生页面多余空白与冗余元素，让同一屏幕内显示更多的 OKR 卡片，全局进度一目了然。
- **⚡️ 详情弹窗预览**：无需离开当前看板页面，点击 OKR 即可在弹窗中快速预览详情，告别繁琐的页面跳转。
- **📝 右侧文档协作**：支持在看板右侧直接唤出并编辑相关文档，实现“边看目标，边写规划”的无缝工作流。
- **🖱️ 操作区域优化**：将右侧/底部的操作按钮巧妙移动至顶部导航栏，释放更多阅读空间。

## 🔒 隐私政策 (Privacy Policy)

**生效日期：2026年**

我们深知隐私对您的重要性。本扩展程序（飞书 OKR 看板增强 / Feishu OKR Mate）的设计初衷是纯粹的效率提升，严格遵循“最小权限”和“数据本地化”原则。我们在此郑重承诺：

### 1. 数据收集与使用
- **绝不收集**：本扩展完全在您的本地浏览器内独立运行，**不收集、不存储、不上传、不共享**您的任何个人身份信息、飞书账号数据、工作文档或 OKR 内容。
- **无后台服务器**：我们没有设立任何用于接收用户数据的外部服务器。您的所有飞书业务数据依然且仅保存在飞书官方服务器与您的本地浏览器中。

### 2. 权限声明
本插件仅申请了维持核心功能运转的最低必要权限：
- `storage` 权限：**仅用于在您的本地设备上保存界面排版的偏好设置开关**（例如：是否开启紧凑模式）。这些设置数据仅保存在您本机的 Chrome 浏览器中。
- `host_permissions` (`*://*.feishu.cn/*` 等)：仅用于在特定的飞书域名下激活排版优化的 CSS 和 DOM 操作脚本，防止在非目标网页运行，确保您的浏览安全。

### 3. 第三方服务
本扩展不包含任何第三方追踪代码（如 Google Analytics、百度统计等），不含有任何广告分发模块。

### 4. 政策更新
如果本扩展在未来增加了需要数据交互的新功能，我们将提前更新本隐私政策，并再次在 Chrome Web Store 提交审核与明确公示。
